# Log Archive Tool

A lightweight, efficient Command Line Interface (CLI) tool built in Bash to automate the compression and archiving of system log directories. 

## Features
- **Validation**: Ensures the provided target directory exists before running.
- **Automated Compression**: Packages logs into a timestamped `.tar.gz` format to save storage space.
- **Logging**: Keeps an execution manifest (`archive_log.txt`) tracking the date, time, and filename of every successfully archived directory.
